Installing Python for Windows

Download the latest version of Python from https://www.Python.org/ and install it on your Windows Operating System.

Installing Numpy

Python supports only single dimensional arrays. To create a multi-dimensional array, we need a special package called Numerical Python package or numpy.

To install this package first, we should go to Command Prompt and then use 'pip' (Python Installation of Packages) as shown below:

C:\> pip install numpy

Installing Pandas

pandas is a package used in data analysis. This package is mostly used by data scientists and data analysts.

To install this package first, we should go to Command Prompt and then use 'pip' (Python Installation of Packages) as shown below:

C:\> pip install pandas

Installing Xlrd

xlrd is package that is useful to retrieve data from Microsoft Excel spreadsheet files. Hence, it is useful in data analysis.

To install this package first, we should go to Command Prompt and then use 'pip' (Python Installation of Packages) as shown below:

C:\> pip install xlrd

Installing Matplotlib

matplotlib is another important package in Python that is useful to produce good quality 2D graphics. It is mostly used for the purpose of showing data in the form of graphs and also for designing electric circuits, machinery parts, etc.

To install this package first, we should go to Command Prompt and then use 'pip' (Python Installation of Packages) as shown below:

C:\> pip install matplotlib
